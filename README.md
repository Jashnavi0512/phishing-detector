# Phishing Email Detector

This project detects phishing messages using Python by analyzing suspicious keywords and links.

## Features
- Detects phishing keywords (urgent, verify, login, etc.)
- Identifies suspicious URLs
- Classifies messages into:
  - Safe
  - Suspicious
  - High Risk

## Example

Input:
Your account is locked, click here https://fakebank.com

Output:
⚠️ High Risk Phishing

## Tech Stack
- Python

## How to Run
1. Run the file:
   python phishing_detector.py
2. Enter a message
3. View result
